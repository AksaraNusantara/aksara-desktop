# Contributing to Aksara Nusantara

Thank you for your interest in contributing!

You don't need to be an expert programmer to help this project.

There are many ways to contribute:

- Improve documentation
- Report bugs
- Suggest new ideas
- Translate documents
- Improve UI design
- Test new features
- Submit code improvements

Every contribution is appreciated.

Please be respectful, constructive, and patient.

Let's preserve Indonesian traditional scripts together.
